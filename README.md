# Online-Cinema
Cinema

### 1. Клонировать репозиторий
```bash
https://github.com/yungvait/Online-Cinema.git
cd Online-Cinema/frontend
```

### 2. Установить зависимости
```bash
npm install
```

### 3. Запустить сервер разработки
```bash
ng serve
```
Приложение будет доступно по адресу:
```bash
http://localhost:4200
```
